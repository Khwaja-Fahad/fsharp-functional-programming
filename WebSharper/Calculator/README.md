# F# Calculator — WebSharper

A functional calculator built with F# and WebSharper SPA.

## Try It Live
🔗 [Click here to use the app](https://khwaja-fahad.github.io/fsharp-functional-programming/Calculator/)

## Screenshot
screenshot.png

## What It Does
- Enter two numbers
- Click +, -, *, / buttons
- See result instantly
- Handles divide by zero safely using F# Option type

## What I Learned
- Multiple pure F# functions
- Option type for safe division (Some / None)
- Pattern matching on operations
- Reactive variables and live UI updates
- WebSharper SPA structure

## How To Run Locally
```bash
cd WebSharper/Calculator
dotnet build
dotnet run
```
Then open: `http://localhost:5000`

## Technologies
- F#
- WebSharper SPA
- .NET SDK 10
