# Recursion in F#

This project demonstrates recursion using a factorial example.

## Topics Covered

- recursive functions
- base case
- recursive case
- call stack concept

## Example

factorial 5

= 5 × factorial 4  
= 5 × 4 × factorial 3  
= 5 × 4 × 3 × factorial 2  
= 5 × 4 × 3 × 2 × factorial 1  
= 5 × 4 × 3 × 2 × 1 × factorial 0  

factorial 0 = 1

Final result:

120
