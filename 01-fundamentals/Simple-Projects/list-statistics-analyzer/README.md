# List Statistics Analyzer

This project analyzes a list of numbers entered by the user.

## Features

- Count numbers
- Calculate sum
- Calculate average
- Find even and odd numbers
- Generate squares of numbers
- Find maximum and minimum values

## Concepts Used

- Lists
- Pipelines
- Map
- Filter
- Functional transformations

## Example

Input:
3 5 8 10 12

Output:
Count: 5
Sum: 38
Average: 7.60
Even numbers: [8; 10; 12]
Odd numbers: [3; 5]
Squares: [9; 25; 64; 100; 144]
