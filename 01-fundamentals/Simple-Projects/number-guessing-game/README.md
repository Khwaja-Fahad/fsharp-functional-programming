# Number Guessing Game

This is a simple console-based number guessing game written in F#.

## Purpose

The program generates a random number between 1 and 100, and the user must guess it.

After each guess, the program tells the user whether the guess is:

- too low
- too high
- correct

The game also counts how many attempts the user needed.

## Concepts Used

- recursion
- conditionals
- user input
- random number generation
- functions

## Example Run

I have selected a number between 1 and 100.

Too low!
