# Functional Banking System

This is a console-based banking system written in F# using a functional programming style.

## Features

- Create account
- Deposit money
- Withdraw money
- Check account balance
- Show all accounts
- Transfer money between accounts

## Concepts Used

- records
- lists
- recursion
- pattern matching
- pipelines
- higher-order functions
- immutable data updates

## Motivation

This project was built to practice functional problem solving in F# while modeling a small real-world banking workflow.

## Example Operations

- Create a new bank account
- Deposit and withdraw safely
- Transfer money between two accounts
- Display all stored accounts
