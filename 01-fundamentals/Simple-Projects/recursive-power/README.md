# Recursive Power Calculator

This project demonstrates recursion in F# by implementing a power calculator.

The program asks the user for:

- a base number
- an exponent

It then calculates the result using a recursive function.

## Concepts Used

- recursion
- functions
- user input
- mathematical operations

## Example

Enter base number: 2  
Enter exponent: 5  

Result:  
2^5 = 32
