# Lists in F#

This project demonstrates important list operations in F#.

## Topics Covered

- creating lists
- adding at front
- adding at end
- accessing by index
- updating by index
- inserting values
- removing values
- head and tail
- length and sum
- map and filter

## Key Learning

F# lists are immutable.  
Any add, update, insert, or remove operation creates a new list instead of changing the original one.
