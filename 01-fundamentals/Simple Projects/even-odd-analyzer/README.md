# Even / Odd Analyzer

This is a simple F# console project built as part of my Functional Programming practice.

## Purpose

The program takes a number from the user and analyzes:

- whether it is **Even** or **Odd**
- whether it is **greater than 10**, **equal to 10**, or **less than 10**

## Concepts Used

- functions
- pattern matching
- conditionals
- user input
- basic operators

## Example Run

Enter a number: 15

Result:
15 is Odd
15 is greater than 10
